### Intro
10년 후에는 데이터를 건드리는 게 기본적인 역량이 될 것이다. 엑셀을 사용하는 것처럼.
더이상 파이썬, R을 통해서 코딩을 하는 교육보다는 패키지로 만들어진 프로그램에 대한 교육의 요구가 더 많다. 현업에서는 이런 툴을 배우고 싶어한다.

오늘 교육은 AccuInsight에 대해서, 이 제품은 어떤 장점이 있고, 데이터 분석에 대한 프로그램을 만드는 부분이 아니라, 데이터 전체적인 상황을 이해하고, 이를 결과를 어떻게 분석하는가에 대한 초점. 내가 확보한 데이터가 무엇이고, 어떤 특성을 가지고 있고, 이런 데이터는 어떤 머신러닝이 적합하고, 어떻게 변환을 시키는 것이 좋을 것인지에 대한 감각이 더 중요. 이를 먼저 공부하고, 프로그래밍에 대한 교육을 해야 함.

현재 기관, 회사에서는 머신러닝을 구축하기에 부적합한 데이터가 98%이다.
데이터 판독이 중요. 데이터 분석을 위한 적절한 데이터를 가지고 있는지 없는지.

(머신러닝을 위해서는 회기 모형이 정의가 되어야 함. 엔지니어적인 사고로 시작하면 100%망한다. 사회전반적인 현상을 어떻게 파악하고, 데이터가 어떻게 도출이된건지에 대한 해석이 되어야 함.)

### 1. Big Data의 이해
AI > 머신러닝 > 딥러닝
AI 는 거대한 하나의 이론적인 개념이지,, 산업을 가리키는 것이 아님.
머신러닝 안에 딥러닝이 포함됨.

인공지능이 가능한 업무
- 반복적인 업무로 누적된 데이터가 많은 영역. (아무것도 없이 인공지능 시스템을 만들 수 없다.) 데이터를 학습해야 인공지능을 만들 수 있다.

시스템적인 사고보다 데이터에 대한 고민을 더 많이 해야, 좋은 머신러닝을 만들 수 있다.

(자연어 처리에서 한글이 어렵다. 한음절 단어는 빠짐? 동음의 단어가 많음..) 한글의 NLP의 문제점이 많은 상태에서 어색한 챗봇이 많이 나오고 있는 현실.

빅데이터의 특징
- 대용량
- 다양한 유형
- 빠른 증가속도

데이터 폐기물 관리자 - 연봉높음. 불필요한 데이터를 버리는..

데이터 저장과 분석 영역 사이의 전문가가 없음.(데이터를 줬다,,<-> 이데이터를 어떻게 분석해) -> 분석할 사람들이 데이터를 미리 잘 파악해서, 데이터를 잘 뽑을 수 있게 해야 함.

빅데이터 역할
- 다수의 시나리오 상황 변화에 유연하게 대처..
- 키워드를 넣으면 어떤 직업을 보여주고,, 어떤 테크를 타야 직업이 되는지를 보여주는 서비스. (중학생의 장래희망, 직업군을 찾는)

빅데이터의 그림을 보고,, 빅데이터의 전처리로 갈건지,, 후처리로 갈건지 내 포지션이 어딘지 생각해볼것. (p.9)

[머신러닝]
설명력 R2.. 설명력이 97%이다
y=ax+b 에서 y값과 x값을 알고 있찌만, a,b값을 모른다. 머신러닝은 a,b값을 찾아가는 과정. 함수에 맞지 않는 x,y값을 버리고 나면,, 맞는게 49%다. 설명력이 49%다. 축에 맞지 않는 데이터는 버려야 한다.[p.10]
머신러닝에서는 설명력이 높은 것을 가지고 머신러닝을 돌려야 함. (설명력이 0.45인걸로 돌리면,, x,y가 상관관계가 없는 데이터를 반을 가지고 머신러닝을 돌리기 때문에 머신러닝 모델이 나올 수 없음.)
- 머신러닝을 만들기 전에, 회기모델이 성립하고, 회기모델의 설명력이 높아야,, 쓰레기로 빠지는 데이터가 적다. 모델이 만들어지면 또,, x,y값을 통과 시킨다.?? (얼마나 x,y가 값이 쌍으로 있는 데이터의 퀄리티가 높은 데이터가 많이 있는지)
- 데이터의 순도.. 불순한 데이터를 제거해서 머신러닝을 돌려야 한다. 이를 판별할 수 있어야 하는 능력이 필요

- 데이터 사이언티스트는 도덕성을 갖추어야 한다. 머신러닝의 위험성. 기준을 조정하면서 원하는 결과를 만들어낼 수 있기 떄문에

[딥러닝]
- 전체 데이터 셋중에 일부분만 넣어줘도 새로운 데이터라고 해석해서 결과를 추출해주거나 분류해줄수 있는 방법이 없을까?
- 학습한 데이터. 기존의 데이터보다 훨씬 많은 학습의 데이터를 가지고 있음. 딥러닝을 만들기 위해서는 거대한 양의 데이터가 필요함.
- 필터라는 개념: 자기 데이터에서 특성을 찾는 것.
- 영상처리의 개념에서 넘어온 것. 지정한 사이즈로 잘라서 1차 필터를 만듬.. 이후에 무한대의 잘라진 필터의 데이터를 placeholder를 통으로 만들어서 넘김. 텐서라는 개념으로 들어감. n차원의 데이터 세트.(n에는 시간등 여러가지가 들어갈 수 잇음)

딥러닝과 머신러닝에서 분석할 수 잇는 타겟데이터는 결국은 정형데이터이다. 이 정형데이터를 만들어가는 전처리가 중요함.
