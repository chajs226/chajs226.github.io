### 이런저런 생각
- IT 부서이다 보니, 아무래도 보안 때문에 좀 조심스러운 부분이 있었다. Read only라도 사내 계정을 받을 수 있다면, 좋을 것 같은데 쉽지가 않다. 디지털팀에서도 그런 사례가 없었기 때문에, 더 조심스러운 것 같다.
- 그리고 프리셉터인 낸시와 다니엘을 통해서 요구 사항을 전달할 수는 있지만, 디지털팀에서 교육이 첨인지라, 아직 체계적이지 못하고, 그들의 업무에 더해져서 나를 지원해줘야했기 때문에, 그들이 나에게 많은 시간을 투자하지 못하는 것 같다. 회의 내용을 공부하고, 자료를 요구해서 혼자 분석하는 시간이 많았다.
