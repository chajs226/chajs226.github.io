## Why?
요즘 해외 연수 준비로 영어 공부를 한다고는 하지만, 생각보다 실력이 쉽게 늘지 않는다. 언어 능력이 썩 좋지 못한 이유도 있겠지만, 제일 큰 문제는 일상 생활에서 영어를 접하는 절대적인 시간과 컨텐츠의 부족이라 생각한다. 한국에서 영어공부를 하는 것의 한계점이다.

그날 배운 표현을 복습해야 하는데, 하루종일 이런저런 일에 치이다 보면 그것도 까먹기 일쑤다. 그래서 내가 공부한 것을 랜덤하게 push 로 '오늘 배운 것을 아직 기억하고 있니? 다시 한번 영어로 얘기해봐' 라고 친절하게 알려주는 App의 필요성을 느꼈다.

그리고 하나의 문장을 다양한 사람이 각자의 표현에 따라 영작한 내용을 볼 수 있다면, 다양한 표현을 익히는데도 도움이 될 것이다.

단어장과 같은 프로그램이지만, 일상생활 중에 좀 더 영어를 접하는 시간을 늘려 주고, 나와 다른 사람들이 만든 표현을 비교해가며 자연스럽게 영어 실력을 키울 수 있는 그런 프로그램이 되었으면 한다.

## What?
1. Home
  - 간단한 통계 보여주기(내가 등록한 문장 갯수, 정답율, 순위 등)
2. Quiz
  - 퀴즈풀기 (랜덤한 문장 불러오기 - 하나씩)
3. My Sentences
  - 퀴즈 문장 등록하기 (제일 위에 등록버튼이 있고, 등록버튼을 누르면 문장을 등록하는 화면으로 전환)
  - 내가 등록한 리스트 보기
    - 리스트 선택시 내가 선택한 정답 보여주기
    - 내가 선택한 정답 아래로 다른 사람의 정답 보여주기
4. History
  - 내가 푼 문장 히스토리 보여주기
  - 리스트 선택 시, 등록된 정답 보여주기 (우선순위에 따라)
5. Settings
  - 로그인
  - 환경설정
    - 푸시 설정
      - 내가 등록한것만 푸시 / 다른 사람이 등록한 것도 푸시 (풀기)
    - 비번 변경

## How?
가장 빠르고 쉽게 만들 수 있는 방법이 필요했다. 웹 프로그래밍을 거의 모르는 상태에서, [생활코딩](https://www.opentutorials.org/course/1)의 웹프로그래밍 강좌를 참고했다. WEBn 강의 중 가장 디테일하게 설명하고 있는 Node.js 를 보면서 따라하면서 만들어보려고 했는데, 아무래도 웹의 기초를 설명해주는 강의이다 보니, 실제 어플리케이션에서 사용할 정도로 만들려면 상당한 노가다가 필요했다.

그러다 아는 개발자분께 자바스크립트부터 공부하는 것도 좋지만, 이미 세상에는 좋은 프레임웤과 도구들이 많이 나와있다며 한번에 가자는 조언을 듣고, 클라이언트 단은 Ionic과 Angular, 서버단은 Python + Django 를 사용하기로 했다. 어차피 뭘로 만들든 모두 새로 공부해야하는 것이니, 왠만하면 나중에 많이 쓰일 것 같은 도구를 선택했다.

처음에는 Youtube에서 Ionic 관련 강의를 찾아보면서 따라해보다가, 체계적으로 배워야 되겠다는 생각에 Udemy의 강좌를 하나 등록을 해서 개발을 진행했다.

1. 생활코딩 NodeJS 편 참고
2. Udemy 강좌: [Ionic 4 - Build iOS, Android & Web Apps with Ionic & Angular](https://www.udemy.com/ionic-2-the-practical-guide-to-building-ios-android-apps/)

매주 시간을 조금씩 내어, 개발 과정에서 배우고 삽질한 것들을 기록으로 남겨두고자 한다.
